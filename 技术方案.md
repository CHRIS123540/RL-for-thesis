#### (1) 常用词

衡量两个回答在核心观点、关键信息点和内容详尽程度上的匹配程度。

四级评分标准：

| 等级 | 描述                            |
| -- | ----------------------------- |
| 0  | 核心结论不一致，甚至存在冲突                |
| 1  | 核心结论一致，但关键要点/细节缺失，语义不完整       |
| 2  | 核心结论一致，大部分要点匹配，细节基本对齐，仅表达略有差异 |
| 3  | 内容完全一致，包括观点、要点和表达方式           |
